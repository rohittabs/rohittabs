Can I request a custom song or file?+

Guitar Pro file details+

The Guitar Pro file you receive is non-editable. You can slow down tempo, loop sections, transpose sections, and mute or unmute parts. Editing notes or structure is not allowed. Editable Guitar Pro files are available at extra cost depending on song duration.
