FAQs

What does duration mean?+

Duration refers to the provided track length.

Can I request a custom song or file?+

Yes. You can request custom songs or files. Please message on WhatsApp before making payment to discuss feasibility and pricing.

Guitar Pro file details+

The Guitar Pro file you receive is non-editable. You can slow down tempo, loop sections, transpose, and mute or unmute parts. Editing notes or structure is not allowed. Editable Guitar Pro files are available at extra cost depending on song duration.
