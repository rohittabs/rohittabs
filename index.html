FAQs

What do I receive?+

Files listed for the selected song including PDFs, tracks, or Guitar Pro files.

What does duration mean?+

Duration refers to the provided track length, not full song length.

Guitar Pro file details+

The Guitar Pro file which you will receive is non editable. You can slow the tempo down, loop sections, transpose sections and mute or unmute parts.

You cannot edit the tab content, title, or structure inside the file.

If you want a Guitar Pro file which can be edited by you, meaning you own the file and can do whatever you want with it, you will have to pay extra for this.

Payment depends upon the song duration which can be discussed in chat.

Can I request a custom song?+

Yes. Message me on WhatsApp before purchasing.

Refund Policy

All sales are final. Once payment is completed and files are delivered, no refunds are provided.


Can I request changes after payment?
Any specific requests or changes must be communicated before the files are delivered. Once delivery is complete, changes are not possible.


What if I selected the wrong song or version?
Please double-check the song, version, instrument, and product code before making payment. Files are delivered exactly as selected from the catalogue.


What if there is a technical issue with the file?
If there is a genuine technical issue such as a corrupted file, it will be fixed or re-sent. This does not qualify for a refund.
